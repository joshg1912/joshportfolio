I've made a few webpages using HTML and Javascript

  • Generative Circles
  • Mondrian Painting
  • Sleep Calculator
  • About Me

- Generative Circles is a generative art piece utilizing Javascript functions. The design was coded using HTML Canvas' arc() function. Then displayed in a grid using nested loops. The Math.random() function was used to display the design in a random order each time the page is refreshed.

-Mondrian painting was coded using HTML Canvas' rect() function. A total of three paintings were drawn, each within its own function. Using Math.random gives the creative control to the computer, generating a different painting with different colors each time the page is refreshed.

-Sleep Calculator utilizes conditionals in Javascript to display a user's sleep debt. If a users actual sleep hours per week is equal to their ideal sleep hours per week, they are on track. If there is a deficiency or surplus, sleep calculator will let the user know how many hours need to be added or taken away.
